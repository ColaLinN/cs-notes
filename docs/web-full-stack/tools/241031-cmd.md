


# 常见CMD 命令


## IO

`tee` 主要作用是从标准输入中读取数据，并将数据同时输出到标准输出和一个或多个文件中。

```cmd
#将ls -l输出保存到 output.txt 中，同时显示在终端
ls -l | tee output.txt
#将ls -l输出追加到 output.txt 中
ls -l | tee -a output.txt
```

## 磁盘空间

#查看文件夹占用空间

du -sh /path/to/directory

#查看指定文件的大小

du -h file_name

#列出当前文件夹下所有文件/夹（包含隐藏文件/夹）的大小，且从小到大排序

du -sh .[!.]* * | sort -h
