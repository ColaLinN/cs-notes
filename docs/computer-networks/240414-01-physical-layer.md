# 物理层 The Physical Layer

## 通信方式

根据信息在传输线路上的传送方向，氛围以下三种通信方式

1. 单工通信：单向传输
2. 半双工通信：双向交替传输
3. 全双工通信：双向同时传输

## 带通调制

带通调制把数字信号转换为模拟信号。

模拟信号是连续的信号，数字信号是离散的信号。
